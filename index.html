VisualPDE Inputs

Minimal examples of using VisualPDEInputs to interact with embedded VisualPDE simulations. Find the source on GitHub.

Sliders

We can use the slider below to control the parameter $a$ in the simulation, which changes the colour and moves the black shape across the domain.

One slider, many simulations

You can use the same slider to control multiple simulations. Here we use two copies of the same simulation, but you can use any number of different simulations that use the same parameter name.

Buttons

We can use custom button elements to play, pause, and reset any VisualPDE simulation without showing the user interface in the simulation.
