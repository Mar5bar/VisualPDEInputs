VisualPDE Sliders

A minimal example of using VisualPDE Sliders to interact with an embedded VisualPDE simulation. Find the source on GitHub.

We can use the following sliders to control the parameters $a$ and $b$ in the simulation, which change the colour and move the black shape around the domain.

Fully specified slider:

A VisualPDE Slider with all options specified:

Partially specified slider:

A VisualPDE Slider with a restricted range and automatically computed step size and minimum.

Minimally specified slider:

A second slider controlling the parameter b. Both sliders can update the simulation, but won't reflect changes to one another.

One slider, many simulations

You can use the same slider to control multiple simulations. Here we use two copies of the same simulation, but you can use any number of different simulations that use the same parameter name.
